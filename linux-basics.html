⌨️ ESSENTIAL COMMANDS

The commands you'll use every single day

📁 Navigation
pwd # where am I?
ls -la # list files
cd /var/log # change dir
cd .. # go up
cd ~ # go home
📄 Files
cat file.txt # view file
cp a.txt b/ # copy
mv a.txt c/ # move/rename
rm file.txt # delete
touch new.txt # create
🔒 Permissions
chmod 755 script.sh
chown user:group file
sudo <command>
# rwx = 4+2+1 = 7
# owner/group/other
🔍 Search & Process
grep "error" log.txt
find / -name "*.conf"
ps aux
kill -9 <PID>
top # live processes
🔑 PERMISSIONS EXPLAINED VISUALLY
-rwxr-xr--
OWNER rwx = 7
read + write + execute
GROUP r-x = 5
read + execute only
OTHERS r-- = 4
read only
⚙️ PACKAGE MANAGEMENT

Installing software on Linux

APT (UBUNTU/DEBIAN)
sudo apt update # refresh package list
sudo apt upgrade # upgrade all packages
sudo apt install nginx # install nginx
sudo apt remove nginx # remove nginx
apt search postgresql # search for package
🔄 PIPES AND REDIRECTION — LINUX SUPERPOWER
BASH
# Pipe: send output of one command to another
ps aux | grep nginx
cat /etc/passwd | sort | head -5

# Redirect output to a file
echo "hello" > file.txt # overwrite
echo "world" >> file.txt # append
ls /etc 2> errors.txt # redirect errors
⚠️ COMMON MISTAKES

What beginners get wrong

❌ MISTAKE 1Running everything as root. Always work as a regular user. Use sudo only when necessary. rm -rf / as root = disaster.
❌ MISTAKE 2Forgetting sudo apt update before install. Always update the package list first, or you'll install an outdated version.
❌ MISTAKE 3Using spaces in filenames. my file.txt is two arguments to a command. Use underscores: my_file.txt.
❌ MISTAKE 4Not reading error messages. Linux error messages tell you exactly what's wrong. Read them carefully — they're not cryptic once you learn them.
💡 Quick Summary — Linux Basics
Linux uses a single root / filesystem — everything is a file, including devices and processes.
Essential commands: ls, cd, cat, cp, mv, rm, grep, find, ps, chmod.
Permissions: rwx for owner/group/others. Numeric: 7=full, 5=read+exec, 4=read only.
The pipe | is Linux's superpower — chain commands together.
Use sudo apt update && sudo apt install to install software on Ubuntu/Debian.
